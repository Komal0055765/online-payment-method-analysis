# Online Payment Method Analysis

## Project Description
This project is based on the analysis of different online payment methods.
The main aim of this project is to understand the usage of various payment modes
such as UPI, Debit Card, Credit Card, Net Banking and Wallet.

The analysis is done using Python with the help of Pandas and Matplotlib libraries.
A self-created CSV dataset is used for this project.

---

## Objectives
- To analyze the usage of different online payment methods
- To perform basic statistical analysis on transaction data
- To visualize the data using bar chart and pie chart

---

## Dataset Information
- Dataset Type: Self-created CSV file
- File Name: digital_payments.csv
- Columns:
  - payment_type: Type of payment method
  - transaction_amount: Amount of transaction

The dataset is manually created based on general observation of online payment usage.

---

## Tools and Technologies Used
- Python
- Pandas
- Matplotlib
- VS Code

---

## Project Files
- online_payment_analysis.py : Python code for analysis and visualization
- digital_payments.csv : Dataset file
- README.md : Project documentation

---

## Steps Performed in the Project
1. Loaded the CSV dataset using pandas
2. Analyzed the usage of payment methods
3. Performed statistical analysis such as total, average, maximum and minimum
4. Created bar chart for comparison of payment methods
5. Created pie chart to show percentage distribution

---

## Output
- Printed dataset and statistical values in terminal
- Bar chart showing usage of payment methods
- Pie chart showing percentage distribution of payment modes

---

## Conclusion
This project helps in understanding the trend of online payment methods.
The use of data visualization makes the analysis easy to understand.
UPI is observed as the most commonly used payment method.
